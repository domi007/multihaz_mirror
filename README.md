# Multiház Mentő/Mirror

A Multiház rendszerben tárolt társasházi dokumentumok lokális biztonsági mentésére szolgáló script. A teljes mappastruktúrát replikálja a `./mirror` mappába. A felhasználó nevet, jelszót és a közösképviselőhöz tartozó egyedi URL-t a fájlban kell beállítani.
